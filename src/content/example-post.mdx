Welcome to your new MDX-powered blog! This post demonstrates some of the cool things you can do with MDX.

## What is MDX?

MDX allows you to write JSX in your Markdown content. It's the best of both worlds: you get the simplicity of Markdown with the power of React components.

```jsx
// Example of a React component inside MDX
function HelloButton() {
  return <button onClick={() => alert('Hello from MDX!')}>Click me!</button>;
}

<HelloButton />
```

### Code Highlighting

Thanks to `rehype-pretty-code`, your code blocks will be beautifully highlighted. Try changing the theme in `tailwind.config.ts`.

```typescript
import React from 'react';
import { useState } from 'react';

interface CounterProps {
  initialCount?: number;
}

const Counter: React.FC<CounterProps> = ({ initialCount = 0 }) => {
  const [count, setCount] = useState(initialCount);

  return (
    <div>
      <p>Current count: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
      <button onClick={() => setCount(count - 1)}>Decrement</button>
    </div>
  );
};

export default Counter;
```
